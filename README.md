# dotfiles

Structure:
```
└── home
    ├── .bash_completions
    ├── .bash_logout
    ├── .bash_paths
    ├── .bash_profile
    ├── .bashrc
    ├── .exrc
    ├── .gitconfig
    └── .gitignore
```

## Setup

```shell
make install
```

## Removal

```shell
make uninstall
```
