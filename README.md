# dotfiles

## Setup/Updating

Link config with
```shell
make link
```

Move scripts (and later update them) with
```shell
make
```

Do both with
```shell
make install
```

## Removal

Remove links with
```shell
make unlink
```

Remove scripts with
```shell
make remove
```

Do both with
```shell
make uninstall
```
